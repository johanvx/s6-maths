Integration

Cadre \(I\) is a real interval and \(\mathring{I} \not= \emptyset\)

Integration of positive functions

I Definitions

Definition Let \(f\) be a positive, continuous almost everywhere function on \(I\). The function \(f\) is integrable on \(I\) if: \[\exists M \in \mathbb{R}^+, \forall [a,b] \subset I, \int_{[a,b]} f \leqslant M.\] In this case, the number \[\int_I f := \underset{[a,b] \subset I}{\sup} \int_{[a,b]} f\] is called integral of \(f\) on \(I\).

Remarks and Notations
  • The set consisting of all segments in \(I\) is written as \(S(I)\).
  • The function \(f\) is integrable on \(I\) if: \[\exists M \geqslant 0, \forall J \in S(I), \int_J f \leqslant M.\]
  • Theoretically, it is necessary to distinguish

    • \[\begin{aligned} \mathop{\int_0^{+\infty} f} \limits_{\textrm{convergence of the integral}} & \textrm{and} & \mathop{\int_0^{+\infty}f} \limits_{\textrm{integrability of }f} \\ \mathop{\sum_{n=0}^{+\infty} u_n} \limits_{\textrm{convergence of the series}} & \textrm{and} & \mathop{\sum_{n \in \mathbb{N}} u_n} \limits_{\textrm{summability (absolute convergence)}} \end{aligned}\]

  • If \(I = [\alpha,\beta]\) is a segment (\(\alpha \lt \beta\)), then \(f\) is always integrable on \(I\) (because it's continuous almost everywhere) and \[\int_I f = \mathop{\underbrace{\int_\alpha^\beta f = \int_{[\alpha,\beta]} f}} \limits_{\textrm{cf. S4}}.\]

    • Indeed,

    • \(f\) is continuous almost everywhere on the segment \([\alpha,\beta]\), so \(f\) is bounded on \([\alpha,\beta]\).

      • \[\forall J \in S(I), \int_J f \leqslant ||f||_{\infty, I} \cdot l(J) \leqslant ||f||_{\infty, I} \cdot (\beta - \alpha).\] Therefore, \(f\) is integrable on \(I = [\alpha,\beta]\).

    • Let \(J \in S(I)\). Since \(f \geqslant 0\), \[\int_J f \leqslant \int_{[\alpha,\beta]} f\] So \[\sup_{J \in S(I)} \int_J f \leqslant \int_{[\alpha,\beta]} f.\]
    • Finally, Since \(I = [\alpha,\beta] \in S(I)\), \[\int_{[\alpha,\beta]} f \leqslant \sup_{J \in S(I)} \int_J f.\]

    In conclusion, \[\int_I f = \int_{[\alpha,\beta]} f\].

Examples

  • \(\displaystyle \int_{(0,1]} \frac{\mathrm{d} x}{\sqrt{x}} = ?\)

    The function \(x \mapsto \dfrac{1}{\sqrt{x}}\) is integrable on \((0,1]\).

    • Theoretically:

      • Let \([a,b] \subset (0,1]\), we have \[\begin{aligned} \int_{[a,b]}\frac{\mathrm{d} x}{\sqrt{x}} & = 2(\sqrt{b} - \sqrt{a}) \\ & \leqslant 2 \sqrt{b} \\ & \leqslant 2. \end{aligned}\] Therefore, \[\forall J \in S((0,1]), \int_J \frac{\mathrm{d} x}{\sqrt{x}} \leqslant 2.\] So the function \(x \mapsto \dfrac{1}{\sqrt{x}}\) is integrable on \((0,1]\) and \[\int_{(0,1]} \frac{\mathrm{d} x}{\sqrt{x}} \leqslant 2.\] In addition, for \(n \in \mathbb{N}^*\), we have \([\dfrac{1}{n}, 1] \in S((0,1])\) and \[\int_{[\frac{1}{n}, 1]} \frac{\mathrm{d} x}{x} \leqslant \int_{(0,1]} \frac{\mathrm{d} x}{x}.\] i.e. \[2 - \frac{2}{\sqrt{n}} \leqslant \int_{(0,1]} \frac{\mathrm{d} x}{x}.\] Yet each term has a (finite) limit when \(n \to \infty\), so \[2 \leqslant \int_{(0,1]} \frac{\mathrm{d} x}{x}.\]

    • Practically:

      • \[\int_a^1 \frac{\mathrm{d} x}{x} = \bigg[2 \sqrt{x}\bigg] \bigg|_a^1 \mathop{\to} \limits_{a \to 0} 2.\]

    • Remark In this example, resolving the problem seems easy because we know a primitive of the function \(x \to \dfrac{1}{\sqrt{x}}\) and the fundamental theorem of calculus.

    In practice, it would never be the case!

  • The function \(x \mapsto \dfrac{e^{-x^2 \sin(x)}}{\sqrt{x} + x^2 \cos^2(x)}\) is integrable on \((0,1]\) and \[\forall x \in (0,1], f(x) \leqslant \frac{1}{\sqrt{x}}.\]

    • Let \(0 \lt a \lt b \leqslant 1\), we have \[\int_{[a,b]} f \leqslant \int_{[a,b]} \frac{\mathrm{d}x}{x} \leqslant 2.\] So the function \(f\) is integrable on \((0,1]\) and \[\int_{(0,1]} f \leqslant 2.\] However, a priori, we are not able to calculate \(\int_{(a,b]} f\).

II Properties

Theorem Let \(f,g \in C_m^0(I,\mathbb{R})\).

  1. If \(f\) and \(g\) are integrable on \(I\), then for each \(\lambda \geqslant 0\) and each \(\mu \geqslant 0\), \(\lambda f + \mu g\) is also integrable on \(I\) and \[\int_I \lambda f + \mu g = \lambda \int_I f + \mu \int_I g.\]
  2. If \(f \leqslant g\) and if \(g\) is integrable on \(I\), then \(f\) is also integrable on I and \[\int_I f \leqslant \int_I g.\]
Proof
  1. Clear since it's the integral on a segment.
  2. By linearity, inequalities between functions and the l.u.b (least-upper-bound) property.

Definition A sequence \(J_n\) is called an exhaustive sequence (of segments) of \(I\) if

  1. \(\forall n \in \mathbb{N}, J_n \in S(I).\)
  2. \(\forall n \in \mathbb{N}, J_n \subset J_{n + 1}.\)
  3. \(\bigcup_{n \in \mathbb{N}} J_n = I.\)

The set consisting of all exhaustive sequences of \(I\) is written as \(S_e(I)\).

Proposition \[S_e(I) \not = \emptyset.\]

Theorem Let \(f\) be a positive, continuous almost everywhere function on \(I\). The following statements are equivalent:

  1. The function \(f\) is integrable on \(I\).
  2. There exists a sequence \(J_n \in S_e(I)\) such that \((\int_{J_n} f)_{n \in \mathbb{N}}\) converges.
  3. For each sequence \(J_n \in S_e(I)\), \((\int_{J_n} f)_{n \in \mathbb{N}}\) converges.

In this case, the limit of \((\int_{J_n} f)\) does not depend on \((J_n)\) and \[\lim_{n \to \infty} \int_{J_n} f = \int_I f.\]

Recall
  • The function \(f \in C_m^0([a,b])\) if there exists \(a = a_0 \lt a_1 \lt \cdots \lt a_n = b\) such that \(f_{|(a_i,a_{i+1})}\) is continuous and can be extended to a continuous function on\([a_i, a_{i+1}]\).
  • The function \(f \in C_m^0(I)\) if \(f\) is continuous almost everywhere on each \([a,b] \subset I\)
Proof
1. \(\implies\) 3.

Let \((J_n) \in S_e(I)\), then \((u_n) = (\int_{J_n} f)_{n \in \mathbb{N}}\) is increasing (since \(f \geqslant 0\) and \((J_n)\) is increasing from the perspective of inclusion).

Yet \(f\) is integrable on \(I\), so \[\forall n \in \mathbb{N}, u_n \leqslant \int_I f = \sup_{J \in S(I)} \int_J f.\] Therefore, \((u_n)\) is bounded from above by \(\int_I f\).

Hence, \((u_n)\) is increasing, bounded from above and so convergent, and \[\lim_{n \to \infty} \leqslant \int_I f.\]

3. \(\implies\) 2.
OK since \(S_e(I) \not = \emptyset\).
2. \(\implies\) 1.

Suppose that there exists an sequence \((K_n) \in S_e(I)\) such that \(\int_{K_n} f\) converges.

Let \([a,b] \in S(I)\). Since \(a \in I = \bigcup_{n \in \mathbb{N}} K_n\), there exists \(n_1 \in \mathbb{N}\) such that \(a \in K_{n_1}\). Likewise, there exists \(n_2 \in \mathbb{N}\) such that \(b \in K_{n_2}\).

Let \(n_0 = \max \{n_1, n_2\}\). As \(a \in K_{n_0}\) and \(b \in K_{n_0}\), we have \[[a,b] \in K_{n_0}\] because \(K_{n_0}\) is an interval.

Therefore, \[\forall n \geqslant n_0, [a,b] \in K_n.\] Hence, \[\forall n \geqslant n_0, \int_{[a,b]} f \leqslant \int_{K_n} f\] since \(f \geqslant 0\).

Yet \((\int_{K_n} f)_{n \in \mathbb{N}}\) converges, so \[\int_{[a,b] f} \leqslant \lim_{n \to \infty} (\int_{K_n} f).\] As a result, \(f\) is integrable on \(I\) and \[\int_I f = \sup_{[a,b] \in I} \int_{[a,b]} f \leqslant \lim_{n \to \infty} \int_{K_n} f.\] In addition, if one of these propositions is satisfied, then for \((J_n \in S_e(I))\):

According to 1. \(\implies\) 3.
\[\lim_{n \to \infty} \int_{J_n} f \leqslant \int_I f.\]
According to 2. \(\implies\) 1.
\[\int_I f \leqslant \lim_{n \to \infty} \int_{J_n} f.\]

Proposition Let \(f \in C_m^0(I,\mathbb{R}^+)\) (or \(f \in C_m^0 (I,\mathbb{K})\)). Let \(c \in I\) and suppose \(I_1 = I \cap (-\infty,c]\) and \(I_2 = I \cap [c,+\infty)\).

The function \(f\) is integrable on \(I\) if and only if \(f\) is integrable on \(I_1\) and \(I_2\).

In this case, \[\int_I f = \int_{I_1} f + \int_{I_2} f.\]

Proof
  • If \(f\) is integrable on \(I\):

    • Let \((J_n) \in S_e(I_1)\) and \((K_n) \in S_e(I_2)\). \[\forall n \in \mathbb{N}, J_n \in S_e(I)\] Hence, \[\forall n \in \mathbb{N}, \int_{J_n} f \leqslant \int_I f.\]

    Yet \((\int_{J_n} f)\) is increasing.

    As a result, \((\int_{J_n} f)\) converges, i.e. \(f\) is integrable on \(I_1\).

    Likewise, \(f\) is integrable on \(I_2\).

    In addition, for \(n \in \mathbb{N}\), by noting \(J_n = [a_n,b_n]\) and \(K_n = [\alpha_n,\beta_n]\) where \(a_n \leqslant b_n \leqslant c \leqslant \alpha_n \leqslant \beta_n\), we have \[\int_{J_n} f + \int_{K_n} f \leqslant \int_{[a_n,\beta_n]} \leqslant \int_I f.\]

    Yet each term has a finite limit when \(n \to \infty\), so \[\lim_{n \to \infty} \int_{J_n} f + \lim_{n \to \infty} \int_{K_n} f \leqslant \int_I f.\] i.e. \[\int_{I_1} f + \int_{I_2} \leqslant \int_I f.\]

  • Reciprocally, if \(f\) is integrable on \(I_1\) and \(I_2\):

    • Let \((J_n) \in S_e(I)\). For \(n \in \mathbb{N}\), let \(K_n = J_n \cap (-\infty,c]\) and \(L_n = J_n \cap [c,+\infty)\). Hence, \[(K_n) \in S_e(I_1) \quad \textrm{and} \quad (L_n) \in S_e(I_2).\] Yet for \(n \in \mathbb{N}\), \[\begin{aligned}\int_{J_n} f & = \int_{K_n} f + \int_{L_n} f \\ & \leqslant \int_{I_1} f + \int_{I_2} f \end{aligned}\] Therefore, \((\int_{J_N} f)\) is bounded from above and so convergent for \((\int_{J_N} f)\) is increasing.

    As a result, \[\int_I f \leqslant \int_{I_1} f + \int_{I_2 f.}\]

III Link with convergence of integrals

a) Recall

Definition Let \(f \in C_m^0 ([a,b), \mathbb{K})\) where \(b \in \mathbb{R} \cup \{+\infty\}\). \(\int_a^b f(x) \mathrm{d} x\) converges if the function \(x \mapsto \int_{[a,x]} f\) has a finite limit on \(b\).

Then \[\int_a^b f := \lim_{\substack{x \to b \\ x \lt b}} \int_{[a,x]} f.\]

Remark Likewise, for \(f \in C_m^0((a,b], \mathbb{K})\), the function \(x \mapsto \int_{[x,b]} f\) is studied.

Theorem Let \(f \in C_m^0((a,b), \mathbb{K})\). The following statements are equivalent:

  1. There exists \(c \in (a,b)\) such that \(\int_a^c f\) converges and \(\int_a^c f\) converges.
  2. For each \(c \in (a,b)\), \(\int_a^c f\) converges and \(\int_c^b f\) converges.

When one of these two propositions is true, \(\int_a^b f\) converges and the value \(\int_a^c f + \int_c^b\) does not depend on \(c\). The value is called \(\int_a^b f\).

By definition, \(\int_a^b f\) converges if one of the two conditions above is satisfied.

Proposition Let \(f, g \in C_m^0(I)\). If \(\int_I f\) and \(\int_I g\) converge, then for each \((\lambda,\mu) \in \mathbb{K}^2\), \(\int_I (\lambda f + \mu g)\) converge and \[\int_I (\lambda f + \mu g) = \lambda \int_I f + \mu \int_I g.\]

b) Link with integrability

Theorem Let \(f\) be a positive, continuous almost everywhere function on \(I\), then \(f\) is integrable on \(I\) if and only if \(\int_I f\) converges.

In this case, by noting \(I = ?a,b?\), \[\int_I f = \int_a^b f(x) \mathrm{d} x.\]

Proof (The case \(I = [a,b)\)) Let \(F: x \mapsto \int_{[a,x]} f\), then \(F\) is increasing since \(f \geqslant 0\). \[\int_a^b f(x) \mathrm{d}x \iff F \textrm{ has a finite limit on } b. \iff F \textrm{ is bounded from above on } I. \iff f \textrm{ is integrable on } I.\] In this case, according to the monotone convergence theorem, \[\int_a^b f(x) \mathrm{d} x = \lim_{x \to b^-} F(x) = \sup_{x \in [a,b)} F(x) = \sup_{x \in [a,b)} \int_{[a,x]} f = \sup_{J \in S_e(I)} \int_J f = \int_I f.\]
